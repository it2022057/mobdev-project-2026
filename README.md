# Android Daily Task Manager

An Android application for managing daily tasks, developed as part of a university assignment.  
The project demonstrates the use of modern Android components, local persistence, background work, and inter-application data sharing.

## Overview

This project focuses on:
- Structured local data storage using Room
- Background processing with WorkManager
- Inter-app communication via Content Provider
- Clean separation between UI, data, and background layers

The application is designed around the concept of **day-based tasks**, automatically handling task lifecycle updates and data cleanup.

## Tech Stack

- **Language:** Java
- **Platform:** Android
- **Minimum SDK:** API 26
- **Database:** Room (SQLite)
- **Background Work:** WorkManager
- **UI Components:** RecyclerView, Activities, Fragments
- **Persistence:** SharedPreferences
- **Inter-App Communication:** Content Provider
- **File Export:** TEXT / HTML (Shared Storage)
- **Maps:** Google Maps (Intent-based)
